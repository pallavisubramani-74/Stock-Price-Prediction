# 📊 Stock Price Prediction using NLP & TensorFlow

## 🔍 Project Description
This project predicts stock price trends by combining **historical market data** with **financial news sentiment analysis**. Using **Natural Language Processing (NLP)** and **TensorFlow**, it extracts insights from text data and integrates them with time-series data to support data-driven decision-making.

---

## 🚀 Key Features
- 📈 Analysis of historical stock price data  
- 📰 Sentiment analysis on financial news articles  
- 🔍 Data cleaning, preprocessing, and EDA  
- 🤖 Predictive modeling using TensorFlow  
- 📊 Visualization of trends and insights  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn  
- **NLP:** NLTK / Scikit-learn (TF-IDF, preprocessing, sentiment analysis)  
- **Modeling:** TensorFlow  
- **Environment:** Jupyter Notebook  

---

## 🔄 Workflow
1. Data Collection (stock prices + financial news)  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Sentiment Analysis using NLP  
5. Feature Engineering (TF-IDF / embeddings)  
6. Model Building using TensorFlow  
7. Evaluation and Visualization  

---

## 📊 Results & Insights
- Identified a relationship between **news sentiment and stock price movement**  
- Positive sentiment often aligns with **price increases**  
- Improved prediction by combining **textual + numerical data**
